Vibes

Growing collection of web vibes. Pick a vibe to start exploring.

This gallery showcases the results of playing with large language models and old ideas: lightweight HTML/JS applications generated entirely via few-shot prompting with LLMs (mainly Gemini 2.5 and 3.0).

For source code, check out the GitHub repository.

  • Thomson Problem
    Find the minimum energy configuration of electrons on a sphere.
  • Triangle Shape Space
    Explore the topology of triangle shapes on a sphere.
  • Sliding Puzzle Chess
    Chess variant inspired by that XKCD comic.
  • Reflecting Ants
    Simulate perfectly elastic ants bouncing inside a mirrored square to reveal intricate trails.
  • Swap Puzzle
    Rebuild the scrambled image by swapping whole rows and columns across expanding grids.
